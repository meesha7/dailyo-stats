About

This project started as a simple tool to visualize data from Daylio, an amazing mood tracker app for Android and iOS. Since the official app doesn't have an option to see all data in one chart, I decided to create a simple script to do exactly that.

Since other folks on Reddit also wanted to get more from their data, I later decided to make it into a very simple web app.

Privacy policy

Using the anonymous upload

This website has been created with privacy in mind, and therefore you can use the anonymous upload, in which case:

  • This website does not use cookies, trackers, ads or anything.
  • Your uploaded files are never stored on the server. Once they're uploaded, they're immediately processed and discarded.

If you want to make sure, you can see the source code on GitHub.

Using an account

Using the site with an account comes with a few additional conditions:

  • Only essential cookies are used: those for keeping you logged in. No other type of cookies is used: no trackers, no ads, no analytics.
  • The files you upload will be processed and discarded, however the data they contain will be saved in a database. Only the data you provide are stored. You can also choose which parts of the uploaded file are stored.
  • Your passwords are stored in a way that nobody can read them. Not even with a database access.
  • You can delete your data anytime. The data will be deleted immediately, with no way of restoring them.
  • Reminder, that this is still a hobby project.

Contributing

Help is always appreciated. You can contribute in the following ways:

  • Report bugs, submit new ideas or ask questions on the Issues page on GitHub
  • If someone has at least a better sense for design than me, please, hit me up
